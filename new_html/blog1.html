Getting started and going deep into Web Development



Getting started and going deep into Web Development

Posted by: Shubham Singhal
Posted on: 17 April 2017

For Beginners:
Start with basic Front End languages : HTML , CSS for making static web pages. HTML : https://www.w3schools.com/html/default.asp CSS : https://www.w3schools.com/css/default.asp Once Basic Front End is done one can start with BackEnd. There are various backend languages (Node.js, Python, PHP etc). I would personally suggest going with PHP initially since it is very similar to C/C++ and one will get to know what backend actually means. For database MySQL along with PHP will be best. And if someone is using nodejs go for MongoDB. Once someone knows any backend language moving on to another platform is pretty simple. PHP : https://www.w3schools.com/php/ http://php.net/manual/en/index.php NODE.js :https://www.w3schools.com/nodejs/ For Those who already have some basic idea of above : Start with BOOTSTRAP / MATERIALIZE CSS which are basically libraries of CSS already written aFond directly available to use. One doesn’t need to remember all the classes(of HTML elements) name of these libraries, just start using them and you will remember the basic and most frequently used classes automatically. BOOTSTRAP:http://getbootstrap.com/ MATERIALIZE CSS :http://materializecss.com/ JAVASCRIPT and JQUERY : It is used to program the behavior of web pages and make your pages more responsive based on various ‘Events’. There is hardly any website today that doesn’t used these language. JAVASCRIPT :https://www.w3schools.com/js/default.asp JQUERY :http://api.jquery.com/ https://www.w3schools.com/jquery/ BACKEND : Once basic structure is known we can move on to using FRAMEWORKS.Web frameworks automate the overhead associated with common activities performed (since there is no need to perform a task again and again ) thus reducing our task a lot . LARAVEL (PHP) : https://laracasts.com/series/laravel-from-scratch-2017 https://laravel.com/docs/5.4/installation DJANGO (python): https://docs.djangoproject.com/en/1.11/ (Docs are very explanatory) Express (Node.js):https://expressjs.com/en/starter/installing.html
